# Chatbot Skill
Use these tools when developing a chatbot
- ElevenLabs Agent

# Refer to WebDev Skill for project structure and other development guidelines

# Goal
Develop a front end of a chat bot that being hooked to Elevenlabs Agent using their API.

# UI/UX
- Make the UI clean and modern.
- Make the UI responsive.
- Make the UI have dominant orange color, and white as secondary color.

# User Flow
1. User open url
2. User press "Begin Conversation"
3. Popup asking for permission to use microphone will appear
4. User grant permission
5. User can start talking to the chatbot
6. User can end the conversation by pressing the end conversation button
7. User can see the conversation history
