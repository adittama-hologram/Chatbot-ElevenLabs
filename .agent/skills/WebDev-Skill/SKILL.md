# WebDev Skill
Use React Js for frontend development

# Docker
Make sure to use docker container

# Project Structure
Use these path when developing a web
src/
├── assets/      # Images, fonts, global styles
├── components/  # Reusable UI elements (buttons, inputs, cards)
├── hooks/       # Custom hooks
├── pages/       # Top-level components that represent specific views/routes (e.g., Home.jsx, Login.jsx)
├── utils/       # Helper functions (formatters, validators)
├── services/    # Code for API calls and external interactions
├── App.jsx      # Main application component
└── index.js     # Entry point (mounts the app to the DOM)

# 404 Page
If route is not found create a 404 page and redirect to the 404 page (make sure inside that page include text that say "Masi Proses Boss").

# Database (if required)
Use MongoDB for database.

# API (if required)
Use REST API for API.

# Assets 
If Assets such as images, icons, etc is required, please use the CC0 assets.

# Gitignore
When developing web app, make sure to add .gitignore file that typically used in projects that use react js
